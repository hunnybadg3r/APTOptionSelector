# APT Option Selector

중재자 패턴을 활용하여 복잡한 옵션 선택 규칙을 효율적으로 관리합니다.

## 주요 기능

- 옵션 간 제약조건 적용
  - 상호 배타적 선택 
  - 선행 조건 확인 
  - 제약 조건 적용 
- 실시간 가격 계산

## 기술 

- .NET 8.0
- WPF (Windows Presentation Foundation)
- C#
